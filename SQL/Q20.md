> "How do you get the **first row (or top N rows)** in **each group**?"

| East   | Alice       | 1000   |

| West   | Carol       | 900    |

You want to **get the top salesperson in each region**, based on `Amount`.

## âœ… Method 1: Using `ROW_NUMBER()` (Best Practice in Modern SQL)

### âœ… Works in: SQL Server, PostgreSQL, Oracle, MySQL 8+, etc.

```sql
SELECT Region, Salesperson, Amount
FROM (
    SELECT *,
           ROW_NUMBER() OVER (PARTITION BY Region ORDER BY Amount DESC) AS rn
    FROM Sales
) AS ranked
WHERE rn = 1;
```

âœ… This returns the **highest amount per region**.

### Works in: MySQL < 8, older systems

```sql

FROM Sales s
WHERE Amount = (
    SELECT MAX(Amount)
    FROM Sales
    WHERE Region = s.Region
);
```

```sql

FROM Sales s
JOIN (
    SELECT Region, MAX(Amount) AS MaxAmount
    FROM Sales
    GROUP BY Region
) m ON s.Region = m.Region AND s.Amount = m.MaxAmount;
```
