### â“ What is an "Error-First Callback"?

In Node.js, **error-first callbacks** are functions where the **first argument is always an error**, and the rest are results (if no error occurred).

Example:

```js
fs.readFile('file.txt', (err, data) => {
  if (err) {
    // handle the error
    return;
  }
  // use the data
});
```
