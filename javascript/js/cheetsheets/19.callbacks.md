## ðŸŽ¯ Callback with Error Handling Convention

```js
function fetchData(callback) {
  setTimeout(() => {
    const error = null; // or new Error('Failed!');
    const data = { id: 1, name: 'John' };
    callback(error, data);
  }, 1000);
}

fetchData((err, data) => {
  if (err) {
    console.error('Error:', err);
  } else {
    console.log('Data:', data);
  }
});
```
